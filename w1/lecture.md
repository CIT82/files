### Assignment Setup

To begin this assignment, open your **private repository** in your development environment and follow these steps to set up your workspace:

1. **Create the Folder Structure:** Inside your private repo, create a new folder named `lectures`.
2. **Create the Subfolder:** Inside the `lectures` folder, create a new folder named `roadmap`.
3. **Initialize the File:** Inside the `roadmap` folder, create a new file named `README.md`.
4. **Add the Title:** At the very top of your `README.md`, create a level 1 markdown heading with the video title: `# The Complete Web Development Roadmap`.
